# 📊 PowerFit India Excel Analysis

## 📌 Project Overview
This project analyzes customer membership data of PowerFit India using Microsoft Excel.

## 🎯 Objective
- Analyze membership duration
- Identify active vs inactive customers
- Perform data cleaning and transformation
- Generate insights using Excel

## 🛠️ Tools Used
- Microsoft Excel
- Basic Data Cleaning
- Formulas (DATEDIF, IF, etc.)

## 📊 Key Tasks Performed
- Data Cleaning and Formatting
- Created new column: Membership Duration (in Months)
- Converted date differences into months (1 month = 30 days)
- Identified Active Members
- Used Excel formulas and functions

## 📈 Insights
- Majority of users are active
- Membership duration varies significantly
- Some data required cleaning due to formatting issues

## 📎 Files
- Excel Dataset: `data/PowerFit_India_Analysis.xlsx`
- Report: `report/PowerFit_India.pdf`
